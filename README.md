# VELORA HUB Discord Bot

بوت Discord باسم **VELORA HUB** لإدارة السيرفر، الترحيب، التذاكر، الاقتراحات، الضريبة، التقييمات، وأوامر الإدارة.

## التشغيل

يتطلب Node.js 24+ لأن البوت يستخدم `fetch` و `WebSocket` المدمجين بدون مكتبات خارجية.

```bash
export DISCORD_TOKEN="BOT_TOKEN"
export DISCORD_CLIENT_ID="APPLICATION_ID"
export DISCORD_GUILD_ID="GUILD_ID"
npm start
```

عند التشغيل يسجل البوت أوامر السلاش داخل السيرفر المحدد في `DISCORD_GUILD_ID`.

## المزايا

- ترحيب تلقائي بإيمبد وصورة عند دخول عضو جديد.
- `/say` لإرسال رسالة بإيمبد أو بدون إيمبد.
- نظام تذاكر دعم فني وشراء عبر بانل أزرار مع زر استلام وزر حذف/إغلاق.
- لوق للتذاكر.
- أوامر إدارة حسب رتب: `Timeout Manager`, `Kick Manager`, `Ban Manager`, `Chat Manager`, `Warning Manager`, `Role Manager`.
- نظام نقاط: كل أمر إداري ناجح يعطي نصف نقطة، و `/points` يعرض النقاط.
- الاقتراحات: يحذف رسالة العضو ويعيد إرسالها بشكل مرتب مع تفاعلات ✅ و ❌.
- الضريبة: يحسب الضريبة تلقائيًا عند إرسال رقم في روم الضريبة.
- التقييم: يحذف رسالة التقييم ويعيد نشرها بإيمبد مرتب مع صاحب التقييم.
- `/summon` لاستدعاء مبرمج أو رسام/مصمم أو مصمم سيرفرات.

## ملاحظات الصلاحيات

يجب تفعيل Privileged Gateway Intents للبوت من لوحة Discord Developer Portal:

- Server Members Intent
- Message Content Intent

ويجب أن يكون للبوت صلاحيات إدارة الرومات، الرسائل، الرتب، الحظر، الطرد، والتايم اوت حسب الأوامر المطلوبة.
